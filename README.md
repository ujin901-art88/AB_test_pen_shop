# A/B Тест для интернет-магазина шариковых ручек

## 📌 О проекте
Проект демонстрирует **полный цикл A/B тестирования**: от бизнес-гипотезы до статистически обоснованного вывода.

**Гипотеза:** Новый дизайн лендинга увеличит конверсию с 5% до 9%.

**Результат:** Конверсия выросла с 5.47% до 9.20% (**+68%**), p-value = 0.0086.

---
